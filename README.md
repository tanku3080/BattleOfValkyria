# BattleOfValkyria
戦場のヴァルキュリアにあるをプラスレズビアン  
  
## 操作方法  
  
 <details> <summary> 挙動操作</summary> 

  + W 前進  
  + S 後進  
  + D 右旋回  
  + A 左旋回  
  + J 砲塔右旋回  
  + L 砲塔左旋回  
  </details>
    
<details> <summary> 攻撃動作</summary>  

  + 右クリック エイム  
  + 右クリック中左クリック 攻撃  
  + R 命中率100%  
  + F 砲塔照準  
  </details>
    
<details> <summary> UI操作</summary>  

  + Enter ターンエンドボタン出現  
  + Space 戦車切替ボタン出現  
  + Q レーダー出現  
  + P ポーズ画面  
  </details>
